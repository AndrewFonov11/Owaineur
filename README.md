# Owaineur

Let me say right away: I am NOT a professional in AI development, especially in C, so I apologize in advance if anyone thinks that the code is written too poorly.

Owaineur - It's a compact neural network of the multilayer perceptron type. It's not a language model, meaning it doesn't process text. It specializes in numbers.

## Characteristics and power

It has 5 inputs, 5 hidden neurons in 1 hidden layer, and 5 output neurons. The interface is console-based, and data is entered directly during program execution.
